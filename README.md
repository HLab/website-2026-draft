# Honey Lab Website

Source for the [Honey Lab](https://hlab.github.io/website-2026-draft) website — a cognitive and neural sciences research group at Johns Hopkins University.

Built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages.

## Local Development

```bash
bundle install
bundle exec jekyll serve --livereload
# Visit http://localhost:4000/website-2026-draft/
```

Requires Ruby 3.3.6 (via rbenv recommended).
